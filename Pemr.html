INTRODUCING TO WEB

Web

Web adalah sebuah aplikasi yang di dalamnya berisi konten multimedia seperti teks, animasi, gambar, dan lain-lain. Menggunakan HTTP (Hypertext Transfer Protocol) untuk menrasfer proses. Untuk mengakses sebuah halaman web (webpage), dibutuhkan sebuah software yaitu browser.


Browser

Browser merupakan sebuah aplikasi untuk mengakses website dan internet. Disaat pengguna meminta halaman web dari sebuah situs web, browser tersebut akan mengambil file dari server web tersebut yang kemudian akan ditampilkan di halaman layar pengguna. Proses tersebut dilakukan oleh sebuah komponen yang disebut web engine. Beberapa contoh browser : Safari, Firefox, Chrome, Edge, Opera.

Bahasa yang digunakan untuk Pemrograman web
HTML CSS JavaScript
Struktur Halaman Web: HTML digunakan untuk mengatur struktur dasar halaman web, termasuk elemen-elemen seperti header, paragraf, gambar, dan tautan. Desain dan Tata Letak: CSS digunakan untuk mempercantik tampilan halaman web dengan menentukan gaya dan tata letak, seperti warna, ukuran font, margin, dan padding. Interaksi Pengguna: JavaScript memungkinkan pengembang untuk menambahkan interaktivitas pada halaman web, seperti validasi formulir, efek hover, dan tindakan berdasarkan input pengguna.
Hyperlinking: HTML memungkinkan pembuatan tautan dan hyperlink yang memungkinkan navigasi antarhalaman web atau sumber daya online lainnya. Responsif Web Design: CSS memungkinkan desain responsif untuk memastikan tampilan yang optimal pada berbagai perangkat dan layar, seperti komputer desktop, tablet, dan ponsel. Manipulasi DOM (Document Object Model): JavaScript dapat digunakan untuk mengakses dan memanipulasi elemen-elemen pada halaman web secara dinamis, memungkinkan pembaruan konten tanpa perlu me-refresh halaman.
Formulir Pengumpulan Data: HTML menyediakan elemen formulir yang memungkinkan pengguna untuk menginput dan mengirimkan data ke server, seperti formulir pendaftaran atau formulir kontak. Animasi dan Transisi: CSS menyediakan kemampuan untuk membuat animasi dan transisi halus, meningkatkan pengalaman pengguna pada situs web.
